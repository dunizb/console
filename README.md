# 随机打印恶搞信息

在浏览器控制台显示一些信息，下面的效果会随机输出，水机数大于20输出

效果举例：
![效果举例：](https://ws1.sinaimg.cn/large/683aa04fly1fynmtfp4xvj20p80d8wfk.jpg)

当随机数小于20时输出18禁止图标
![18禁](https://ws1.sinaimg.cn/large/683aa04fly1fynp3pzb7yj2054030t8n.jpg)

安装：
```
npm install er-console
或
yarn add er-console
```
导入
```
import 'er-console'
```
